## What this is
An end-to-end Trustworthy AI research framework applied to electrical calibration data from CSIR-NPL.  
Combines semantic knowledge graphs, conformal uncertainty quantification, explainable AI (SHAP), and graph neural network anomaly detection — all accessible via an interactive Streamlit dashboard.

## Framework overview
Calibration Certificates (ODT/DOC)
↓
01_extract_data.py       → calibration_data.csv (31 pts, 4 instruments)
02_build_kg.py           → electrical_metrology.ttl (799 RDF triples)
02b_validate_kg.py       → SHACL validation (Conforms: True)
03_ml_pipeline.py        → ml_results.csv + model_bundle.pkl
04_shap_analysis.py      → shap_values.csv (KernelExplainer XAI)
05_gum_comparison.py     → gum_comparison.csv (GUM vs conformal PI)
06_gnn_anomaly.py        → gnn_results.csv (GCN autoencoder)
07_autoencoder.py        → autoencoder_results.csv (tabular AE)
08_multitask.py          → multitask_results.csv (3-task sklearn)
↓
dashboard/app.py         → 7-tab Streamlit dashboard

## Novelty vs existing work
- First application of **conformal prediction** as a UQ layer for electrical calibration data
- **GCN autoencoder** anomaly detection on calibration graphs (no prior art in metrology)
- **SOSA + QUDT + PROV-O + custom EMO** ontology combination for metrological knowledge graphs
- **SHAP KernelExplainer** reveals physics-consistent feature importance (domain > instrument type >> indicated value)
- **Multi-task learning**: one model → ratio error + distribution type + uncertainty class simultaneously

## How to run

### Prerequisites
```bash
python -m venv .venv
.venv\Scripts\activate        # Windows
pip install -r requirements.txt
```

### Run the full pipeline
```bash
python run_all.py
```

### Launch the dashboard
```bash
streamlit run dashboard/app.py
```
Open `http://localhost:8501`
