    Data schemas — all CSV files
    calibration_data.csv (primary — 31 rows, 4 instruments)
    Column	Type	Meaning
    cert_id	string	Certificate identifier
    instrument	string	Instrument name
    measurement_domain	string	current or voltage
    indicated_value	float	Indicated/nominal value (A or kV)
    measured_value	float	Actual measured value (A or kV)
    coverage_factor_k	float	GUM coverage factor
    expanded_uncertainty_pct	float	GUM expanded uncertainty U (%)
    ratio_error_pct	float	(measured-indicated)/indicated × 100
    standard_uncertainty_pct	float	u = U/k (%)
    range_position	float	Normalised position in instrument range (0–1)

    ml_results.csv
    Column	Type	Meaning
    ml_predicted_ratio_error_pct	float	Model predicted ratio error
    conformal_lower/upper_95pct	float	95% conformal prediction interval bounds
    conformal_halfwidth_pct	float	PI half-width — the 'ML uncertainty'
    gum_expanded_U_pct	float	Traditional GUM expanded uncertainty U
    loo_residual_pct	float	Leave-one-out residual (how hard this point was)

    gnn_results.csv  |  autoencoder_results.csv
    Column	Type	Meaning
    gnn_anomaly_score	float	GNN reconstruction MSE — higher = more anomalous
    gnn_is_anomaly	bool	True if score > 75th percentile threshold
    ae_score	float	Autoencoder reconstruction MSE
    ae_anomaly	bool	True if score > 75th percentile threshold

    multitask_results.csv
    Column	Type	Meaning
    mt_pred_ratio_error_pct	float	Task 1: predicted ratio error
    mt_pred_nongaussian	int	Task 2: 1=non-Gaussian (k>2.0), 0=Gaussian

    Data loading pattern
    ROOT = Path(__file__).parent.parent  # goes up from dashboard/ to project root
    OUTPUTS = ROOT / 'outputs'
    @st.cache_data
    def load_all():
        d = {}
        for name in ['calibration_data','ml_results','gum_comparison',
                    'shap_values','gnn_results','autoencoder_results','multitask_results']:
            try: d[name] = pd.read_csv(OUTPUTS / f'{name}.csv')
            except: d[name] = None
        return d

    Live calculator — model loading
    import joblib
    @st.cache_resource
    def load_model():
        return joblib.load(OUTPUTS / 'model_bundle.pkl')

    def predict_live(bundle, indicated, domain, instrument):
        dm = {'current':0.0,'voltage':1.0}
        im = {'AC High Current Source':0.0,'Clamp Meter':1/3,'HV Probe':2/3,'HV Breakdown Tester':1.0}
        X = np.array([[indicated, dm[domain], im[instrument]]])
        pred = bundle['model'].predict(X)[0]
        return pred, bundle['q']   # prediction, PI half-width
